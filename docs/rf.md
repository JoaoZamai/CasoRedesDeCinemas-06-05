## RF01 - Login
O usuário deve realizar o login a partir de Email e Senha, que devem ser válidos.

## RF02 - Visualiação do Filme/Sessão
O usuário poderá ver qual filme está sendo exibido em determinada sessão.
Além disso, poderá ver quais sessões estão em andamento.

## RF03 - Número de Clientes
Cada sessão terá registro do número de clientes presentes.

## RF04 - Filmes
Cada filme terá suas informações como Idade, gênero, duração, elenco, diretores ..., que será armazenado em um banco de dados.

## RF05 - Cadastro de Clientes
Caso seja vontade, o cliente pode criar um registro no Cinema para receber possíveis benefícios.

## RF06 - Estoque
O usuário poderá visualizar a quantidade de produtos comestíveis, não comestíveis, dentre outros, disponíveis no estabelecimento.
